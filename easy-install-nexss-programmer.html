Install Nexss Programmer 2.2.0 Open Source

Through NPM - Linux, MacOS, Windows

note: Linux and MacOS are still in the development stage so not all languages are supported.

npm install @nexssp/cli -g

or

npx @nexssp/cli

LEGACY INSTALLERS

Through Git (NodeJS 12/14 must be installed)

git clone --depth=1 https://github.com/nexssp/cli.git
cd cli
chmod +x nexss.js
ln -s $(pwd)/nexss.js /usr/bin/nexss

Windows

Run below code using Powershell:

Set-ExecutionPolicy Bypass -Scope Process -Force; iwr -useb nexss.com/installProgrammer | iex

If you are having errors about tls/ssl channel please run first:

[Net.ServicePointManager]::SecurityProtocol = [Net.SecurityProtocolType]::Tls12

Linux

We tested Nexss Programmer on: Windows and Linux: Alpine, Amazon, Arch, CentOS, Debian, Fedora, Mint, Oracle, Rhel, Suse Leap, Suse TumbleWeed, Ubuntu. MacOS: soon

Run this script on your Linux machine (Legacy Installer - see `npm i @nexssp/cli or npx @nexssp/cli`): Nexss Programmer Linux Installer

Note: You don't need to download Nexss Programmer separately and just run installer.
