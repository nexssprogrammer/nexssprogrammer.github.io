Sequences

Nexss PROGRAMMER allow to use sequences to have multiple files section. Eg

From Nexss Programmer 2.1.1+, sequence names can be used as regular expressions (capturing groups and named groups.). Please see below

nexss start --seq=myseq1

sequences:
  default: # this sequence fill run first (if there is no files section in the config)
    - name: boot.py
  open_file:
    - name: src/open.tcl
    - name: Data/Show --showFields=fileData
    - name: Template --name=myname.jade
  read_data:
    - name: src/readdata.r
  www:
    - name: src/serverWWW.rs
  (?<page>.*).hnxs:
    - name: Template src/views/pages/${page}.pug # Use named groups from regular expressions as params.
  about.pug:
    seq: default # Example of usage of another sequence in sequence 2.1.1+
  ^about(.)(.*): # Regular expression Capturing group. Data will be passed as nxsParam_0, 1, 2 (0 is a sequence), so you can use them in the templates, modules etc.
    seq: default
  ^2about(?<found1>.)(?<found2>.)(?<found3>.*): # Data will be passed as found 1, 2, 3 (0 is a sequence name), so you can use them in the templates, modules etc.
    seq: default
