Programming languages in Nexss Programmer List of the languages and contact details

In the Nexss Programmer we have implemented many programming languages and technologies and these are not maintained by Nexss so please contact the respective maintainers. if you have any questions or issues related to the language itself.
All websites with the contact information are listed below.
If you're the maintainer of a the language or technology not listed / implemented in the Nexss Programmer and would like your project listed here don't hesitate to contact us.
